About mediaTUM

mediaTUM is an open source software product for large scale image, document and video archiving and retrieval written in Python. You can see a live system here: mediaTUM @ tum.de

Features of mediaTUM

  • Fast metadata and fulltext indexing and search
  • Support for document, video and image file formats (pdf, ps, jpeg, png, tif, swf)
  • Decentralised Administration
  • Extensible, rule-based, access control
  • OAI export
  • OpenAIRE compliant
  • user and user group management
  • Plugins
